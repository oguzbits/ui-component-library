# Luminance: Darkness as a Native Medium

In our system, color is not a static hex code. It is an **opacity step** on top of a "Void Black" (`#000000`) canvas. This ensures that every layer of the UI feels "stacked" and meaningful.

## The "Chiseled" Border
Borders in this system are never "Grey." They are translucent white dividers that react to whatever background is behind them.

- **Primary Border**: `rgba(255, 255, 255, 0.08)`
- **Subtle Border**: `rgba(255, 255, 255, 0.05)`
